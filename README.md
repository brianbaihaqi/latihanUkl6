Program ini digunakan untuk menjumlahkan dua buah matriks yang dimasukkan oleh pengguna. Pertama, pengguna diminta untuk menentukan ukuran matriks (jumlah baris dan kolom). Kemudian, program meminta input elemen-elemen dari matriks A dan matriks B. Setelah kedua matriks diisi, program melakukan penjumlahan elemen yang bersesuaian dari kedua matriks dan menyimpannya dalam matriks hasil. Terakhir, hasil penjumlahan ditampilkan dalam bentuk matriks ke layar.
